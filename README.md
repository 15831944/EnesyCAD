# EnesyCAD Project
Công cụ quản lý các Lisp mở rộng trong AutoCAD dành cho Kỹ sư

Các tính năng
=======
Các tính năng chính:
* Quản lý các lệnh mặc định của AutoCAD
* Quản lý các Lisp, các lệnh của Lisp
* Tạo các danh sách lệnh yêu thích
* Tra cứu nhanh đến các lệnh
